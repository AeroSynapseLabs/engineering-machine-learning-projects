# Machine Learning Projects (Python)

This repository contains a collection of machine learning projects implemented in Python.
The projects focus on practical applications of machine learning methods, including regression analysis and convolutional neural networks for image classification.

The goal of this repository is to demonstrate practical experience in:

* data preprocessing and feature engineering
* machine learning model development
* neural network implementation with PyTorch
* model evaluation and visualization
* working with real-world datasets

---

## Implemented Topics (Up to 16.03.2026)

### Regression Models

Implementation and comparison of several regression techniques for predictive modeling of aircraft Take-Off Weight, including:

* Linear Regression
* Ridge Regression
* Lasso Regression


The project demonstrates the impact of regularization and model complexity on prediction accuracy.

---

### Convolutional Neural Networks (CNN)

Implementation of multiple CNN architectures for aircraft image classification using the FGVC-Aircraft dataset.

Key components include:

* dataset preprocessing and bounding box cropping
* data augmentation
* comparison of different CNN architectures
* training monitoring (accuracy and loss curves)
* confusion matrices and classification metrics
* Grad-CAM visualizations for model interpretability
